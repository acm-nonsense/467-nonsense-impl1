Scraping Piazza

Piazza famously does not have a publicly accessible API. This left us with options of either reverse engineering the API (very difficult) or using Selenium to automate a browser (very slow). We chose the Selenium appoach and found that we had to babysit the script to ensure that it would progress as a result of how the Piazza UI worked. A couple of hours later, we had the data we needed


Natural Language Processing

We used AlchemyAPI for our sentiment Analysis. We had 11 thousand requestions that needed to be made. This far over the free account ratelimit for AlchemyAPI. We got around this by simply submitting the requests slowly over a period of time.
